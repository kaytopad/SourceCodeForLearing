要件定義
・ゲームの目的：
二人対戦形式でボードゲームの「九九じゃん」をコンソール画面で遊べるゲームを実装します
これにより、数学的思考力やゲーム戦略を楽しみながら向上させることができます。

・動作環境：
使用言語：C言語もしくはC++
プラットフォーム：コンソールアプリケーション
操作窓口：キーボードから入力

使用するデータ構造：
1.タイル
typedef struct {
    int 段; // 段番号 (2~9)
    int 値; // 方程式の値
} Tile;

2.プレイヤー
typedef struct {
    Tile 手持ち[4]; // プレイヤーの手持ちタイル
    int スコア;    // 勝利値
} Player;

3.山札
4.捨て札

画面設計：
1，開始画面
・ゲーム名を表示
・プレイヤー名を入力
2，ゲーム中画面
・各プレイヤーの手持ちタイルと行動を表示
・行動の入力を受け付け
3，終了画面
・勝者の情報を表示
・もう一度ゲームを行うかの確認
